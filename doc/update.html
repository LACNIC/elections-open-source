{{ "Actualización del Sistema de Elecciones" }}

El proceso de actualización del sistema consiste de 3 pasos: obtener la última versión, ejecutar scripts de base de datos (si es necesario) y publicar la nueva versión. A continuación un detalle de estos pasos.


Descarga de última versión:

En el directorio donde haya clonado el repositorio, ejecute git pull para descargar la última versión.


Ejecución de scripts de base de datos:

Algunas versiones incluyen cambios a nivel de base de datos. En estos casos, se proveen los scripts en los Release notes.

Importante: debe ejecutar, en orden, todos los scripts desde su versión actual hasta la versión que está instalando.


Empaquetado y publicación:

Los procedimientos para empaquetado y publicación son los mismos que cuando se instala por primera vez. Se tienen dos opciones:

  • Copiar los artefactos provistos en el repositorio, o los generados con Maven, al directorio deployments de su instalación de Wildfly. Puede encontrar más detalles sobre esta opción aquí.
  • Usar el playbook Ansible provisto para publicación de los artefactos. Puede encontrar más detalles sobre esta opción aquí.
