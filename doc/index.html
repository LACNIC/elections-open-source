El Sistema de Elecciones es una iniciativa open source, desarrollado con el framework Wicket de Apache. Utiliza Wildfly como servidor de aplicación y PostgreSQL como motor de base de datos.

A continuación se presentan dos alternativas para la instalación y despliegue del aplicativo.

La primera es utilizando Ansible para ejecutar playbooks que realicen todo el trabajo. En el siguiente enlace encontrarán detallado todo el proceso para diferentes sistemas operativos.

En caso no este familiarizado con Ansible o desee revisar el paso a paso manual por favor continúe con el siguiente enlace para los pasos detallados y requisitos para la instalación manual.
