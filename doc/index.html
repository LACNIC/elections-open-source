{{ "Instalación y configuración del Sistema de Elecciones" }}

El Sistema de Elecciones es una iniciativa open source de LACNIC. Está desarrollado en Java usando el framework Wicket de Apache, corre sobre WildFly como servidor de aplicaciones y usa PostgreSQL como motor de base de datos.

Para la instalación y configuración inicial del sistema se tienen dos opciones: automática y manual. En ambos casos se ofrecen instructivos paso a paso para los sistemas operativos Ubuntu Server 18, Debian 10 y CentOS/RedHat 7:

  • Instalación automática: se definieron tareas Ansible para el aprovisionamiento completo del ambiente, puede seguir aquí las instrucciones paso a paso.
  • Instalación manual: si prefiere ejecutar manualmente la instalación, puede seguir aquí las instrucciones paso a paso.
